# SimpleWebNavigation
这是一个基于PHP给个人使用的网址导航，所有链接均可以在txt文件中添加与删除而不需要修改代码

修改 `data.txt` 就能修改导航卡片

格式：
```
box
分类名称
iconfont名称
链接名
链接地址
链接名
链接地址
.....
.....
endbox
```

举个栗子：
```
box
常用
iconfont icon-tool
Get it on GitHub
https://github.com/KrunkZhou/SimpleWebNavigation
Maps
https://www.google.ca/maps/
Translate
https://translate.google.com/
GitHub
https://github.com/
BiliBili
https://www.bilibili.com/
endbox
```
